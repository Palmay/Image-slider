
OPEN index.html TO START USING THE APP!



To add image batch you create a new folder in the Characters folder and name it the name you want it to show in the website, all the images inside to start from 1 in
the order you want them to play for exemple (1.jpg comes before 2.jpg and so) 
 
To add music just drag an mp3 file to the music folder and change the name to one of the existing music(you can delete the existing music if you want).

 The speed in the wesite changes the delay between images, you can put anything from 0 to about 10000. 
 
 if you dont put width and hight the default size of all the images will show on the website. 
 
 Enjoy!!!
